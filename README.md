# Luminic Realism

Artistic portfolio website for Chilean painter Jorge Riveros and his project **Luminic Realism**.

## Files

- `index.html` - main website structure
- `styles.css` - visual system, responsive layout, lightbox, and atmospheric styling
- `script.js` - animated light filaments, gallery lightbox, zoom, pan, and interactions
- `images/` - artwork and portrait images
- `cv/` - selected exhibitions PDF

## Publishing on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save.

GitHub Pages will publish the site at:

`https://your-username.github.io/your-repository-name/`
